# FTamas77.github.io

Minimal static GitHub Pages landing site for Causal AI work.

## Files

- `index.html` - page structure
- `styles.css` - visual styling
- `site-data.js` - content you will most likely edit first

## First edits

1. Open `site-data.js`
2. Replace the publication placeholder with the real paper title and URL
3. Swap repository entries if you want different public repos on the front page

## Publish on GitHub Pages

1. Create a new GitHub repository named `FTamas77.github.io`
2. Push these files to the `main` branch
3. GitHub will serve the site automatically at `https://ftamas77.github.io/`

## Local preview

Open `index.html` directly in the browser, or from this folder run:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.
